## 📊 Superstore Business Performance Dashboard
## 📌 Project Overview

This project delivers an end-to-end retail analytics solution using Python and Power BI.
It combines exploratory data analysis (EDA) with interactive dashboards to uncover insights across product performance, operational efficiency, and managerial accountability.

## 🧠 Business Questions Addressed

Which products and sub-categories are driving losses?

How do discounts and base margins impact profitability?

How does shipping speed influence returns and profit margins?

Where should management focus to improve performance?

## 🛠 Tools & Technologies

Python (Pandas, Numpy, Matplotlib, Seaborn) – data cleaning & EDA

Power BI – interactive dashboards & DAX measures

Excel – raw data source

Git & GitHub – version control and project delivery

## 📊 Dashboard Pages

1️⃣ Executive Overview

High-level KPIs and trends summarizing overall business performance.

2️⃣ Product View

Product and sub-category profitability, discount impact, and margin distribution.

3️⃣ Operations View

Shipping efficiency, delivery delays, and their relationship with returns and profitability.

4️⃣ Manager View

Manager-level performance, return rates, and consistency across regions.

## 🚀 Key Insights

Strong overall profitability with moderate margins.

Technology is the primary profit driver.

Furniture is a margin drag.

Returns are well controlled.
